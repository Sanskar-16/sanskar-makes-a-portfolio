## 🎨 Immersive & Interactive Features

### 📦 The Custom Funko Pop Collectible Case
*   **Interactive 3D Tilt Case**: An elegant, responsive 3D card layout featuring a custom-rendered Funko Pop likeness of Sanskar (short textured black hair, chevron mustache, jaw stubble, silver hoop ring piercing, dressed in a black Reply crewneck).
*   **Hardware-Accelerated Rotations**: Smooth vector mouse-tracking tilt effects that simulate a premium packaged toy box case in physical space.

### 🧗 Playful Organic Passion Doodles
*   **Hand-Drawn Style Stick Doodles**: Features custom animated illustrations of Sanskar's favorite pastimes:
    *   *Bouldering*: Alternate pulsed holds with an interactive carabiner sliding on hover.
    *   *Modular Synth*: Vibrating soundwaves, rotating control knobs, and patch cords.
    *   *Cooking*: Tilting skillet, chopped vegetables, and dynamic stove flames.
    *   *Cycling*: Rotating pedal crank, moving chains, and spinning spokes.
*   **Stop-Motion Animation Style**: Powered by stepped `@keyframes hand-drawn-wiggle` CSS steps that mimic organic notebook sketches.

### 💻 Live GitHub Commits Ledger Feed
*   **Real-time GitHub REST API Integration**: Queries the `sanskar-portfolio` repository commits directly.
*   **Dynamic Elapsed Time Stringing**: Automatically parses timestamps to display context-rich intervals (e.g., *Just now*, *3m ago*, *2h ago*).
*   **Failsafe Fallback Routing**: Gracefully falls back to public profile push events and static mock ledger registries under rate-limit caps.
*   **Interactive Diffs**: Clicking any commit ledger line routes you directly to the raw code diff on GitHub.

### 🖨️ Professional Click-to-Print CV
*   **One-Click Print System**: Both major "DOWNLOAD CV" triggers run standard browser-level print engines (`window.print()`).
*   **Dynamic Print CSS Boundary Isolation**: Harnesses clean `@media print` selectors to hide all interactive display interfaces and reveal a gorgeous, high-contrast, black-and-white, single-page professional resume formatted to print on a single sheet of A4 paper.

## 🛠️ The Tech Stack & Architecture

*   **Core Framework**: [Next.js 16 (App Router)](https://nextjs.org/) using Turbopack compiler.
*   **Language**: [TypeScript](https://www.typescriptlang.org/) for static type-safety.
*   **Design & Styling**: Custom **Vanilla CSS System** implementing HSL tokens, glassmorphic filters, and modular synth hardware aesthetics (avoiding bulky utility utilities).
*   **Interactive Canvas**: Saturated watercolor paint trail canvas layer with pre-feathered HTML5 radial gradient particle decay.
*   **Packaging**: Multi-stage lightweight **Docker containerisation** (Node 20-alpine).
*   **Hosting & Delivery**: **Google Cloud Run** serverless container hosting (auto-scaling 0 -> N).
*   **CI/CD Pipeline**: **GitHub Actions** compiling, building, and deploying on push.

## 📦 Local Development

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/Sanskar-16/sanskar-makes-a-portfolio.git
    cd sanskar-makes-a-portfolio
    ```

2.  **Install Dependencies**:
    ```bash
    npm install
    ```

3.  **Boot Development Server**:
    ```bash
    npm run dev
    ```
    *Open [http://localhost:3000](http://localhost:3000) to view the canvas patchbay.*

4.  **Production Compile Build Check**:
    ```bash
    npm run build
    ```
